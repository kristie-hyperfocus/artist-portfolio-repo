# Artist Portfolio Studio — Example Sites

Three fully built example artist portfolio websites demonstrating distinct design aesthetics. Built as client demos to show what's possible before committing to a direction.

**Live demo:** [View on GitHub Pages](https://YOUR-USERNAME.github.io/artist-portfolio-demo/)

## Important Note

> **These are demonstration sites only.** Elena Voss, Cora Ashby, and Sol Maren are fictional artists. All artwork images are placeholders from Unsplash. These sites are designed to demonstrate layout, typography, interaction style, and aesthetic direction — not to represent real artists or their work.

---

## What's Inside Each Site

Every example includes the following sections:

- **Navigation** — Fixed sticky nav, smooth scroll, glassmorphism on scroll
- **Hero** — Full-viewport entry point (different interaction per site)
- **Gallery** — Interactive artwork showcase with hover reveals
- **About** — Portrait, biography, career stats
- **Collections** — Works grouped by series
- **Recent Works** — Latest pieces with enquiry prompts
- **Inspirations** — Artist statement and influences
- **Social Feed** — Instagram grid placeholder
- **Footer** — Newsletter, navigation, social links

All sites include:
- Fully responsive (mobile, tablet, desktop)
- SEO meta tags + Open Graph
- Accessible markup (ARIA labels, semantic HTML, keyboard navigation)
- Smooth scroll-reveal animations via IntersectionObserver
- No frameworks — pure HTML/CSS/JavaScript

## Deploying to GitHub Pages

1. Push this repo to GitHub (see setup below)
2. Go to **Settings → Pages**
3. Under *Source*, select **Deploy from a branch**
4. Choose `main` branch, `/ (root)` folder
5. Click **Save** — the site will be live at `https://YOUR-USERNAME.github.io/REPO-NAME/`
